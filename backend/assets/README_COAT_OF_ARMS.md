# Equatorial Guinea Coat of Arms

## Required File:
`coat-of-arms-equatorial-guinea.png`

## Download Instructions:

### Option 1: Wikimedia Commons (Recommended - Highest Quality)
1. Visit: https://commons.wikimedia.org/wiki/File:Coat_of_arms_of_Equatorial_Guinea.svg
2. Click "Download" on the right side
3. Choose PNG format at 1024px or higher resolution
4. Save as `coat-of-arms-equatorial-guinea.png` in this folder

### Option 2: NicePNG (Direct PNG)
1. Visit: https://www.nicepng.com/ourpic/u2w7i1r5o0i1q8y3_equatorial-guinea-flag-hd-wallpaper-coat-of-arms/
2. Download the PNG image
3. Save as `coat-of-arms-equatorial-guinea.png` in this folder

### Option 3: ClipartMax (High Resolution)
1. Visit: https://www.clipartmax.com/middle/m2i8i8G6H7m2Z5K9_cotton-tree-equatorial-guinea-coat-of-arms/
2. Download the transparent PNG (1879x2400)
3. Save as `coat-of-arms-equatorial-guinea.png` in this folder

## Image Requirements:
- Format: PNG with transparent background
- Recommended size: 512x512 pixels minimum
- Max size: 2048x2048 pixels
- Aspect ratio: Square or close to square

## Description:
The coat of arms features:
- Silver shield with silk-cotton tree (Ceiba pentandra)
- Six golden stars representing mainland + 5 islands
- National motto: "UNIDAD • PAZ • JUSTICIA"

## If Image Not Found:
The PDF service will use a text placeholder: `[ESCUDO DE GUINEA ECUATORIAL]`
This allows the system to function while you download the proper image.

## After Download:
1. Place the downloaded image in this folder
2. Rename it to exactly: `coat-of-arms-equatorial-guinea.png`
3. Restart the backend service (PM2 restart ministerial-api)
4. The PDF will now show the official coat of arms
